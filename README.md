# BadmintonScore Legal

BadmintonCount 앱의 법적 문서(개인정보처리방침, 이용약관) 전용 저장소입니다.

## 파일 구조

- `index.html` : 법적 문서 홈
- `privacy/index.html` : 개인정보처리방침
- `terms/index.html` : 이용약관
- `assets/style.css` : 공통 스타일
- `.nojekyll` : GitHub Pages 정적 배포용 설정

## GitHub Pages 설정

1. 저장소 `Settings` → `Pages`
2. `Build and deployment`에서 Source를 `Deploy from a branch`로 선택
3. Branch를 `main`, Folder를 `/ (root)`로 선택
4. 배포 완료 후 아래 URL 사용

- Legal Home: `https://jiseobkim.github.io/BadmintonScore_Legal/`
- Privacy Policy: `https://jiseobkim.github.io/BadmintonScore_Legal/privacy/`
- Terms of Service: `https://jiseobkim.github.io/BadmintonScore_Legal/terms/`

## App Store 반영 위치

- 앱 내 Paywall의 약관/개인정보 링크
- App Store Connect의 `Privacy Policy URL`
- (구독 사용 시) App Store Connect의 `Terms of Use` 관련 링크
